## Базовая компиляция

Чтобы скомпилировать один .java файл (на Arch Linux) используется
программа `javac`.
```
javac File.java  
```

После у вас появится файл File.class 
и его можно активировать так.
```
java File // без .class  
```
В итоге: 
### `javac` <- компилятор
---
### `java` <- виртуальная машина

## Функции

Функции в java скорее методы не могут существовать вне класса
нельзя тупо обьявить функцию ему нужен класс или же хоть какой-то
namespace.

---
Функции в `Java`:

```
<модификаторы>  <возвращаемый тип>  <название> 
(...Параметры) 
{
    ...тело
}
```

Например:
```
void sayHello(){
    
    return "Hello";
}

```
---
```
int sum(int a,int b){
    
    return a+b;
}
```


## Произвольное количество параметров
Можно указать произвольное количество параметров одного типа
при помощи `...`

* Пример:
```
public void printNumbers(String dimeter,int... numbers){
    for(int n : numbers){
        System.out.print(n);
        System.out.print(dimeter);
    }
}
```
Элемент `numbers` называется `varargs`
Можно указать сколько угодно чисел 
> *ВАЖНО* нужно указать после обычных параметров.
```
printNumbers("-",12,1,1,2,4,23,13);
```
> *ВАЖНО* `varargs` нужно указывать последним

## Перегрузка
Функции можно перегружать.То есть разные параметры но одно название
```
void print(String... lines){...} // Используется когда мы указываем строки


void print(int... numbers){...} // Когда использованы числа
```

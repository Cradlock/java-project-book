## Структура java - кода 

`Java` -  это обьектно-ориентированный язык. Вся система строится на обьектах и классах.

В языке используется несколько основноых видов файлов.

* .jar: zip архив в котором упакованы ваш код и медиа ресурсы


* .java: искодный код


* .class: скомпилированный байт код который может прочитать JVM 

* .properties|.gradle|.yaml ...: это уже файлы сборки и настройки проекта о них потом


`JVM` - это спец программа которая нужна чтобы исполнять .class обьекты.

В java процесс компиляции и выполнения работает по другому.

```
    file.java 
        |
        |
    file.class 
        |
        |
    JVM (read file.class) -> action

```
То есть .class это все равно что .exe или .elf
> Именно из-за этого код на Java может запустится где угодно потому что код один а JVM разные поэтому один и тот же .class может запускатся на разных ОС и устройствах


## Скорость
в JVM настроен JIT компилятор,который многократно ускоряет .class код превращая в чистый машинный код,из-за этого Java это гибрид 
между компилируемым и интерпретируемым.Также код может оптимизироватся в runtime.

## Проект
В java есть понятие `пакет`

* `package`: Это папка которая хранить несколько классов.И обычно отвечает за отдельный функционал


## Код 
В `.java` все состоит из инструкций `statement` это минимальная
часть кода:
```
System.out.println("Hello world");
```
В конце надо ставить `;`.
---
Код java начинается с точки старат а именно функции `main`.В главном 
публичном классе. 
```
# Programm.java 

public class Programm{ // Название файла должно совпадать с классом
    public static void main(String args[]){
        ... // Код 
    }

}
```

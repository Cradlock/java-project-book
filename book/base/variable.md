## Переменные 

`Java` типизированный язык, в переменных заранее указывается тип 
данных.

> Синтаксис

<Тип данных> <название> = <значение>;
```
int variable = 12;
```
---
Название - нельзя использовать:
* Уже зарезервированные имена 

* Цифра в начале

* Знаки пунктуации и пробелы
---


## Константы
> Можно задать значение один раз
```
final int LIMIT = 1;
```

## var 
Если тип данных очевиден и компилятор сам может догадатся
используем `var` 
> Пример
```
var b = 12; // сам поймет что это int 
```

## null
`null` - указатель на пустоту

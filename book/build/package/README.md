## package

`package` - Система папок которая решает следуюшие проблемы:

> Конфликт имен

> Управление доступом


## Код
```
package com.example.org;

public class Site{
    ...// 
}

```

## Структура 
Название пакето обязaно совпадать с иерархией папок на диске. 
```
package com.example.org;

public class Site{}
```
папки
```
src/com/example/org/Site.java 
```
